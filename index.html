FEATURED PROJECT

BusyBee Movers

What it is: A side-scrolling co-op game about moving a TV through a city. Each character can only see in front of them so players have to communicate to safely get through obstacles. Supports LAN / online play. This game was created for the ArtCenter 2021 Game Jam. It won the Best in Theme award and placed second in the community vote.
What I did (implemented):
  • Networked obstacles
  • UI programming
  • Checkpoints
  • Bugfixes
  • Level layout/design
Team size: 4
Project time period: 48 hours (March 2021)
