FEATURED PROJECT

BusyBee Movers

What it is: A side-scrolling co-op game about moving a TV through a city. Each character can only see in front of them so players have to communicate to safely get through obstacles. Supports LAN / online play. This game was created for the ArtCenter 2021 Game Jam. It won the Best in Theme award and placed second in the community vote.
What I did (implemented):
  • Networked obstacles
  • UI programming
  • Checkpoints
  • Bugfixes
  • Level layout/design
Team size: 4
Project time period: 48 hours (March 2021)

ABOUT ME

Hi there! My name is Colin and I make games using C# and C++ in engines like Unity and Unreal. I am also familiar with the Unreal blueprint system. I have a knowledge of AI, UI, system, and network programming. I also enjoy playing around with shaders and the Godot engine.

OTHER PROJECTS

Eldritch Hotel

What it is: Eldritch hotel is a 2D game about the player being an eldritch horror and escaping from the hotel they were summoned to. The player has various abilities to make the guests of the hotel go insane. The goal of the game is to power up and escape the hotel before the investigators defeat the player.
What I did (implemented):
  • NPC system with Finite State Machine
  • A* pathfinding for NPCs
  • Room editor to make design easier
  • UI programming
  • Ability implentation
Team size: 7
Project time period: 6 weeks (March 2021 - April 2021)

Party Cards

What it is: Party Cards is a mobile local multiplayer game aimed to be played, unsurprisingly, in party situations. I learned about and did paired programming with a recent Champlain College alumni for Party Cards. This project was still in the early stages so most of the work was for the backend and UI.
What I did (implemented):
  • Flinging cards
  • Touch controls
  • Deck creation and card filters
  • Different style sheets
  • Model-view-viewmodel implementation
  • Tons of bug squashing
Team size: 2
Project time period: 6 months (January 2021 - July 2021)

Focus Pocus

What it is: Focus Pocus is a game where the player has to balance completing the task they are assigned and keeping boredom down by playing with other objects around the classroom. This project was aimed to educate younger students about ADHD who may not know about it.
What I did (implemented):
  • Task system for players to interact with
  • Physics for throwing objects around
  • Shaders
  • UI Programming
  • Objects utilizing Unity's hinges
Team size: 7
Project time period: 4 weeks (February 2021 - March 2021)

Daybreak

What it is: Daybreak is a 2D puzzle platformer about a vampire safely getting back to her coffin by blocking sunlight and gliding across platforms. The player can control bats to block light coming directly from the sun or filtering through windows.
What I did (implemented):
  • Lighting with Unity URP
  • UI Programming
  • Character movement (both vampire and bat)
  • Obstacle implementation (lights, moving platforms, buttons)
  • Sound implementation
  • Fixed merge conflicts for Unity scenes
Team size: 7
Project time period: 3 weeks (January 2021 - February 2021)
